1.发表一篇博客

  • cd到博客所在的本地仓库
  • 执行hexo new 文章名称
  • 在source/_posts文件夹中找到.md文件进行编辑
  • 依次执行hexo clean ==清理
  • hexo generate == 生成文章
  • hexo deploy == 发布文章

2.博客中如何添加一张图片

方法一:

  1. 在 _config.yml 的 post_asset_folder 已设为 true
  2. cd到hexo目录下,命令输入“hexo new post 文章名”时都会在source/_posts文件夹下自动创建这样的一个文件夹,将图片放置到自动创建的文件夹下

  3. 编辑文案时使用markDown语句输入图片

方法二:

  1. 直接使用markDown语句输入图片

    ! [img caption] (img url) //url一般为网络地址
